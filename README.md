# InsightBoard – Enterprise Cloud Middleware & UI Integration (Frontend)

A fully responsive, React-based dashboard UI built with Tailwind CSS and Recharts to match enterprise-quality visual standards.

## 🚀 How to Run

```bash
npm install
npm run dev
```

## Features

- Bold headline with filters
- KPI Cards with % change
- Line, Bar, Pie Charts
- Session Data Table
- Responsive Tailwind layout
